## 2. Google Apps Script 설정

1. 구글 시트에서 **확장 프로그램 → Apps Script** 클릭
2. `Code.gs` 파일 내용을 붙여넣기
3. `SHEET_ID`에 구글 시트 URL의 ID 입력
   - URL: `https://docs.google.com/spreadsheets/d/여기가_ID/edit`
4. **배포 → 새 배포 → 웹 앱** 선택
   - 실행 사용자: 나
   - 액세스: 모든 사용자
5. 배포 후 나오는 URL을 복사

## 3. index.html 연결

1. `index.html`에서 `API_URL` 값을 배포 URL로 변경
2. `USE_DUMMY`를 `false`로 변경

## 4. 배포 (Netlify)

1. https://app.netlify.com 접속
2. index.html 파일을 드래그앤드롭으로 업로드
3. 완료! URL 공유하면 끝
